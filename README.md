# Calculadora básica en Python

Este proyecto es una calculadora básica que funciona en la consola, desarrollada en Python.
Utiliza programación orientada a objetos y permite realizar operaciones aritméticas simples con validación de errores.

## Funcionalidades
- Suma
- Resta
- Multiplicación
- División
- Manejo de errores al dividir entre cero
- Menú interactivo en consola
- Opción para salir del programa

## Tecnologías utilizadas
- Python 3
- Programación orientada a objetos
- Manejo de excepciones

## Estructura del proyecto
calculadora-python/
├── calculadora.py
└── README.md

## Ejecución del programa
1. Entrar a la carpeta:
cd calculadora-python

2. Ejecutar:
python calculadora.py
